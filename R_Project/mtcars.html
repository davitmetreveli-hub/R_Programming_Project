Project Overview

This project analyzes the mtcars dataset, which contains performance and design specifications for 32 car models from 1974 Motor Trend magazine.
The goal is to explore relationships between variables, compare groups, and build a simple regression model to understand what affects fuel efficiency (miles per gallon, mpg).
